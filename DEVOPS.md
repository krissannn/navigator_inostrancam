# DevOps документация проекта "Навигатор для иностранных студентов"

## О проекте
Проект "Навигатор для иностранных студентов" помогает студентам ориентироваться в корпусах университета и важных документах.

---

## 1. Локальный запуск проекта (для разработчиков)

### Требования
- Установленный Docker (https://docker.com)
- Установленный Git

### Команды для запуска

```bash
# 1. Клонировать репозиторий
git clone <ссылка на репозиторий>

# 2. Перейти в папку проекта
cd название-проекта

# 3. Запустить все сервисы
docker compose up -d

# 4. Проверить, что всё работает
docker compose ps
