## 💻 Примеры кода

### JavaScript / TypeScript (Fetch API)

```javascript
// Получить все корпуса
async function getBuildings() {
  try {
    const response = await fetch('http://localhost:8000/api/buildings');
    const data = await response.json();
    console.log('Корпуса:', data);
    return data;
  } catch (error) {
    console.error('Ошибка:', error);
  }
}

// Получить аудитории корпуса
async function getBuildingRooms(buildingId) {
  try {
    const response = await fetch(
      `http://localhost:8000/api/buildings/${buildingId}/rooms`
    );
    const data = await response.json();
    console.log('Аудитории:', data);
    return data;
  } catch (error) {
    console.error('Ошибка:', error);
  }
}

// Получить шаги адаптации
async function getSteps() {
  try {
    const response = await fetch('http://localhost:8000/api/steps');
    const data = await response.json();
    console.log('Шаги:', data);
    return data;
  } catch (error) {
    console.error('Ошибка:', error);
  }
}

// Получить статьи шага
async function getStepArticles(stepId) {
  try {
    const response = await fetch(
      `http://localhost:8000/api/steps/${stepId}/articles`
    );
    const data = await response.json();
    console.log('Статьи:', data);
    return data;
  } catch (error) {
    console.error('Ошибка:', error);
  }
}
```

### React (с использованием useEffect)

```jsx
import { useState, useEffect } from 'react';

const API_BASE = 'http://localhost:8000';

function App() {
  const [buildings, setBuildings] = useState([]);
  const [steps, setSteps] = useState([]);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    // Загрузка корпусов
    fetch(`${API_BASE}/api/buildings`)
      .then(res => res.json())
      .then(data => {
        setBuildings(data);
        setLoading(false);
      })
      .catch(err => {
        console.error('Ошибка загрузки корпусов:', err);
        setLoading(false);
      });

    // Загрузка шагов
    fetch(`${API_BASE}/api/steps`)
      .then(res => res.json())
      .then(data => {
        setSteps(data);
      })
      .catch(err => {
        console.error('Ошибка загрузки шагов:', err);
      });
  }, []);

  if (loading) return <div>Загрузка...</div>;

  return (
    <div>
      <h1>Корпуса:</h1>
      {buildings.map(building => (
        <div key={building.id}>
          <h2>{building.name}</h2>
          <p>{building.address}</p>
        </div>
      ))}

      <h1>Шаги адаптации:</h1>
      {steps.map(step => (
        <div key={step.id}>
          <h2>{step.title}</h2>
        </div>
      ))}
    </div>
  );
}
```

### Vue.js

```vue
<template>
  <div>
    <h1>Корпуса:</h1>
    <div v-for="building in buildings" :key="building.id">
      <h2>{{ building.name }}</h2>
      <p>{{ building.address }}</p>
    </div>
  </div>

<script>
export default {
  data() {
    return {
      buildings: [],
      loading: true
    }
  },
  async mounted() {
    try {
      const response = await fetch('http://localhost:8000/api/buildings');
      this.buildings = await response.json();
      this.loading = false;
    } catch (error) {
      console.error('Ошибка:', error);
      this.loading = false;
    }
  }
}
</script>
```

## 🔧 Настройка CORS

CORS уже настроен на бекенде! 

В `main.py` установлено:
```python
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],  # Разрешены все домены
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

- ✅ Ты можешь делать запросы с `localhost:3000`, `localhost:5173` и т.д.

## 📊 Структура данных

### Building (Корпус)
| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `id` | integer | Уникальный идентификатор |
| `name` | string | Название корпуса |
| `address` | string | Адрес |
| `lat` | float | Широта (для карты) |
| `lon` | float | Долгота (для карты) |

### Room (Аудитория)
| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `id` | integer | Уникальный идентификатор |
| `name` | string | Номер аудитории |
| `building_id` | integer | ID корпуса (внешний ключ) |
| `floor` | integer | Этаж |
| `capacity` | integer | Вместимость |

### Step (Шаг адаптации)
| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `id` | integer | Уникальный идентификатор |
| `title` | string | Название шага |
| `icon` | string | Иконка (название) |
| `order` | integer | Порядок сортировки |

### Article (Статья)
| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `id` | integer | Уникальный идентификатор |
| `step_id` | integer | ID шага (внешний ключ) |
| `title` | string | Заголовок |
| `content` | string | Текст на русском |
| `content_en` | string | Текст на английском |
| `order` | integer | Порядок сортировки |

## 🐛 Обработка ошибок

### Возможные коды ответов:

| Код | Значение | Что делать |
| :--- | :--- | :--- |
| `200` | OK | Запрос успешен |
| `404` | Not Found | Эндпоинт или ресурс не найден |
| `422` | Validation Error | Неверный формат данных |
| `500` | Server Error | Ошибка на сервере (сообщи бекенду) |

### Пример обработки ошибок:

```javascript
async function fetchData() {
  try {
    const response = await fetch('http://localhost:8000/api/buildings');
    
    if (!response.ok) {
      throw new Error(`HTTP error! status: ${response.status}`);
    }
    
    const data = await response.json();
    return data;
    
  } catch (error) {
    console.error('Ошибка при загрузке данных:', error);
    // Покажи пользователю сообщение об ошибке
    return [];
  }
}
```

## 🆘 Если что-то не работает

| Проблема | Решение |
| :--- | :--- |
| **CORS error** | Проверь, запущен ли сервер |
| **404 Not Found** | Проверь URL эндпоинта |
| **Пустой ответ []** | В базе нет данных (попроси тимлида заполнить) |
| **Connection refused** | Сервер не запущен (скажи бекенду) |
| **Нужен новый эндпоинт** | Скажи бекенду, какие данные нужны |
