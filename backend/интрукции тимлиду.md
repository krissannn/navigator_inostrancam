# Инструкция для контент-менеджера / тимлида
## Как наполнить базу данных навигатора

---

## 📋 Что нужно заполнить

Для работы навигатора нужны три типа данных:

1. **Корпуса (Buildings)** — здания университета с координатами
2. **Шаги адаптации (Steps)** — этапы (0. Въезд, 1. Заселение и т.д.)
3. **Статьи (Articles)** — подробная информация для каждого шага

---

## 🚀 Способ 1: Через Swagger (Рекомендуется)

### Шаг 1. Открой документацию

Перейди по ссылке:
- **Локально**: http://localhost:8000/docs
- **На сервере**: (ссылка будет после деплоя)

Ты увидишь список всех доступных запросов.

---

### Шаг 2. Добавь корпуса

1. Найди раздел **`POST /api/buildings`**
2. Нажми **Try it out**
3. В поле **Request body** вставь JSON:

```json
{
  "name": "Главный корпус",
  "address": "ул. Мира, 19",
  "lat": 56.8389,
  "lon": 60.6057
}
```

4. Нажми **Execute**
5. Внизу появится ответ с данными (включая `id`)

**Примеры корпусов для заполнения:**

```json
{
  "name": "Учебный корпус №2",
  "address": "ул. Мира, 2",
  "lat": 56.8400,
  "lon": 60.6100
}
```

```json
{
  "name": "Библиотека",
  "address": "ул. Ленина, 1",
  "lat": 56.8350,
  "lon": 60.6000
}
```

---

### Шаг 3. Добавь шаги адаптации

1. Найди раздел **`POST /api/steps`** (если есть) или используй прямой SQL
2. Если эндпоинта нет — скажи бекенд-разработчику, чтобы добавила

**Примеры шагов:**

```json
{
  "id": 0,
  "title": "Въезд и Миграционная карта",
  "icon": "plane",
  "order": 0
}
```

```json
{
  "id": 1,
  "title": "Первичный заезд",
  "icon": "car",
  "order": 1
}
```

```json
{
  "id": 2,
  "title": "Заселение в общежитие",
  "icon": "home",
  "order": 2
}
```

---

### Шаг 4. Добавь статьи

1. Найди раздел **`POST /api/articles`**
2. Нажми **Try it out**
3. Вставь JSON:

```json
{
  "step_id": 0,
  "title": "Как заполнить миграционную карту",
  "content": "Миграционная карта — это документ, подтверждающий законность пересечения границы.\n\nГде получить:\n- Самолет: Выдают стюардессы в полете или на стойках перед паспортным контролем в Кольцово.\n- Поезд/Автобус/Машина: Выдают пограничники на КПП.\n\nКак заполнять:\n1. Цель визита: ОБЯЗАТЕЛЬНО подчеркнуть «УЧЕБА» (EDUCATION)\n2. Принимающая сторона: Пишем «УрФУ» или «UrFU\"\n3. Срок пребывания: Дата въезда + дата окончания визы",
  "content_en": "Migration card is a document confirming legal border crossing.\n\nWhere to get:\n- Plane: Flight attendants give it or at counters before passport control at Koltsovo.\n- Train/Bus/Car: Border guards give it at checkpoints.\n\nHow to fill:\n1. Purpose of visit: MUST underline \"EDUCATION\"\n2. Receiving party: Write \"UrFU\"\n3. Period of stay: Entry date + visa expiration date",
  "order": 0
}
```

4. Нажми **Execute**

## 📊 Способ 2: Через DBeaver (Для продвинутых)

Если ты умеешь работать с базами данных:

1. Установи **DBeaver** (https://dbeaver.io)
2. Подключись к PostgreSQL:
   - Host: `localhost`
   - Port: `5432`
   - Database: `navigator`
   - Username: `postgres`
   - Password: (спроси у бекенд-разработчика)
3. Открой таблицу `buildings`, `steps` или `articles`
4. Правой кнопкой → **Edit Data**
5. Вставляй данные прямо в ячейки

## 📝 Готовые тексты для заполнения

### Шаг 0: Въезд и Миграционная карта

**Статья 1: Где получить миграционную карту**

```
Миграционная карта выдается бесплатно при пересечении границы.

ГДЕ ПОЛУЧИТЬ:

✈️ Самолет:
- Стюардессы выдают в полете
- ИЛИ на стойках перед паспортным контролем в аэропорту Кольцово

🚆 Поезд/Автобус/Машина:
- Выдают пограничники на контрольно-пропускном пункте (КПП)

📋 ВАЖНО:
В аэропорту Кольцово сейчас часто стоят автоматические принтеры, 
которые печатают карту на основе данных паспорта. 
ПРОВЕРЬ распечатку, прежде чем отходить от окна пограничника!
```

**Статья 2: Как заполнять миграционную карту**

```
ГЛАВНЫЕ ПРАВИЛА ЗАПОЛНЕНИЯ:

1️⃣ ЦЕЛЬ ВИЗИТА (самое важное!):
   Студент ОБЯЗАН подчеркнуть или вписать «УЧЕБА» (EDUCATION)
   ⚠️ Любая другая цель (частная, туризм, работа) — это автоматический 
      отказ в регистрации в УрФУ!

2️⃣ ПРИНИМАЮЩАЯ СТОРОНА:
   Пишем «УрФУ» или «UrFU»

3️⃣ СРОК ПРЕБЫВАНИЯ:
   - Дата въезда: (ставится автоматически)
   - Дата выезда: дата окончания визы ИЛИ +90 дней для безвизовых стран

❗ ОШИБКИ:
Если в автоматической распечатке ошибка в цели визита — 
ИСПРАВЛЯЙ НЕМЕДЛЕННО на месте, пока не отошел от окна!
```

## 🆘 Если что-то не работает

| Проблема | Решение |
| :--- | :--- |
| **Swagger не открывается** | Проверь, запущен ли сервер |
| **Ошибка 422** | Проверь JSON на правильность (кавычки, запятые) |
| **Данные не сохраняются** | Скажи бекенд-разработчику |
| **Нужен новый эндпоинт** | Скажи бекенд-разработчику |
